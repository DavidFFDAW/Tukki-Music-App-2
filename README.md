# Tukki-Music-App

Esta aplicación con la que podrás escuchar canciones y tener tus propias playlists a la vez que buscar canciones y usuarios.

## Instalación

#### Frontend -- React
```javascript

cd frontend
npm install
npm run start
+ configurar proxy en el package.json

```
